    - [ ] from_dict_template(dict, **kwargs)
      Creates a dataclass based on a dict template

      from_dict_template({'x': int, 'y': int}, name='Point', **kwargs)

      equivalent to

      @dataclass(**kwargs)
      class Point:
          x: int
          y: int
